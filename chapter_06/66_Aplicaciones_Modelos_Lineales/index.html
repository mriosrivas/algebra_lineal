6. Ortogonalidad y Mínimos Cuadrados

6.6 Aplicaciones a Modelos Lineales

Una tarea común en ciencia e ingeniería es analizar y entender relaciones entre diferentes cantidades que varían. Esta sección describe una variedad de situaciones en las que los datos se emplean para construir o comprobar una fórmula que predice el valor de una variable como función de otras variables.

En esta sección se elige la notación que comúnmente se utiliza en el análisis estadístico de datos científicos y de ingeniería. En vez de escribir $A\boldsymbol{x} = \boldsymbol{b}$, se escribe $X\boldsymbol{\beta} = \boldsymbol{y}$, donde $X$ es la matriz de diseño, $\boldsymbol{\beta}$ es el vector de parámetros y $\boldsymbol{y}$ es el vector de observaciones.
